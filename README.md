# 💰 Leanz – Loan Management System

## 📌 Overview
Leanz is a web-based Loan Management System designed to streamline the process of managing loan applications, customer records, loan approvals, and repayment tracking. The platform helps users apply for loans online while enabling administrators to efficiently manage and monitor loan-related activities.

## 🚀 Features
- User Registration and Login
- Customer Profile Management
- Loan Application Submission
- Loan Approval and Rejection
- Loan Status Tracking
- Repayment Monitoring
- Secure Data Management
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Java

### Database
- MySQL

## 🎯 Objectives
- Simplify loan application processing.
- Reduce manual paperwork.
- Improve customer experience.
- Maintain accurate loan records.
- Provide secure and efficient loan management.

## 👥 User Roles

### Customer
- Register and Login
- Apply for Loans
- View Loan Status
- Track Repayment Information

### Administrator
- Manage Customer Records
- Review Loan Applications
- Approve or Reject Loans
- Monitor Loan Repayments
